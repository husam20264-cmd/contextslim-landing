## About ContextSlim

ContextSlim is a local context engineering CLI that reduces token waste and builds task-specific context packs for AI coding tools.

## Key Features

- **Repo Indexing** — Scans your entire project, extracts file metadata and important functions
- **Task-Based Selection** — Understands your task and selects only relevant files
- **Smart Relevance Scoring** — Multi-factor scoring system for optimal context selection
- **Token Budget Reports** — Shows before/after tokens with savings percentage
- **Local-First** — All processing happens locally, no code leaves your machine
